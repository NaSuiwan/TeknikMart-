# TeknikMart-
Final Project for Algorithms and Programming Subject (May 2026)

Teknik Mart adalah aplikasi marketplace berbasis digital yang dirancang untuk memfasilitasi kegiatan jual-beli barang dan jasa di lingkungan mahasiswa. Saat ini, proses transaksi antar mahasiswa masih banyak dilakukan secara informal melalui grup chat, sehingga pencarian produk tidak efisien dan informasi mudah terlewat. Tidak adanya platform khusus membuat mahasiswa kesulitan untuk membandingkan harga dan berkomunikasi langsung dengan penjual di satu tempat. Oleh karena itu, Teknik Mart hadir sebagai solusi yang memungkinkan mahasiswa untuk menawarkan produk, mencari barang yang dibutuhkan, serta terhubung langsung dengan sesama mahasiswa secara cepat dan terstruktur.
